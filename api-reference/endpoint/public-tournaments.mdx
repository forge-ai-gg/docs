## Overview

Retrieve a paginated list of tournaments using API key authentication. This endpoint is designed for machine-to-machine integrations, dashboards, and services that don't require user authentication.

<Note>
This endpoint uses API key authentication instead of user JWT tokens. Perfect for data pipelines, monitoring dashboards, and automated systems.
</Note>

## Authentication

Use the `X-API-Key` header with your API access key:

```bash
X-API-Key: your-api-key
```

<Warning>
Protect your API key. Never expose it in client-side code or public repositories.
</Warning>

## Query Parameters

Same filtering options as the authenticated endpoint:

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `pageIndex` | integer | `0` | Page number (0-indexed) |
| `pageSize` | integer | `10` | Items per page (1-50) |
| `orderBy` | string | `createdAt` | Sort field |
| `order` | string | `desc` | Sort direction |
| `search` | string | — | Search by name or description |
| `visibility` | string | — | Filter: `public` or `private` |
| `type` | string | — | Filter by tournament type |
| `status` | string | — | Filter: `upcoming`, `active`, or `completed` |

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://app.forgeai.gg/api/v1/public/tournaments?status=active" \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json"
```

```javascript Node.js
const response = await fetch(
  'https://app.forgeai.gg/api/v1/public/tournaments?status=active',
  {
    headers: {
      'X-API-Key': process.env.FORGEAI_API_KEY,
      'Content-Type': 'application/json'
    }
  }
);

const { data, pagination } = await response.json();
```

```python Python
import os
import requests

response = requests.get(
    "https://app.forgeai.gg/api/v1/public/tournaments",
    params={"status": "active"},
    headers={
        "X-API-Key": os.environ["FORGEAI_API_KEY"],
        "Content-Type": "application/json"
    }
)

result = response.json()
```

</CodeGroup>

```json

      "id": "tourn_abc123",
      "name": "Weekly Championship",
      "status": "active",
      "currentParticipants": 42,
      "maxParticipants": 100,
      ...
    }
  ],
  "pagination": {
    "pageIndex": 0,
    "pageSize": 10,
    "totalCount": 15,
    "totalPages": 2

```
