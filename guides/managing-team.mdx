## Building Your Team

### Starting Out

**Phase 1: Your First Agent (Levels 1-10)**
- Create one Agent (recommend Fighter for beginners)
- Focus on learning the game mechanics
- Enter open tournaments to gain experience
- Reach Level 10 to unlock core abilities

**Phase 2: Expand to Two Classes (Levels 10-20)**
- Create a Agent from a different class
- Learn the differences between classes
- Enter class-specific tournaments
- Build both Agents to Level 20

**Phase 3: Complete Your Team (Levels 20+)**
- Create your third Agent from the remaining class
- Now you have all three classes
- Enter diverse tournament types
- Specialize each Agent for their class strengths

## Team Management Strategies

### Level Distribution

<AccordionGroup>
  <Accordion title="Balanced Levels" icon="scale">
    Keep all Agents at similar levels. This ensures you can enter the same tier tournaments with any Agent.
    
    **Best for**: General play, flexibility
  </Accordion>
  <Accordion title="Specialized Levels" icon="target">
    Focus on leveling one Agent to high levels first, then level others. The high-level Agent can enter elite tournaments.
    
    **Best for**: Competitive players, maximizing one Agent
  </Accordion>
  <Accordion title="Class Specialization" icon="users">
    Level Agents of one class higher than others. Dominate class-specific tournaments.
    
    **Best for**: Players who prefer specific strategies
  </Accordion>
</AccordionGroup>

## Common Mistakes

<AccordionGroup>
  <Accordion title="Neglecting Diversity" icon="exclamation-triangle">
    Don't create all Agents from one class. You'll miss opportunities in class-specific tournaments.
  </Accordion>
  <Accordion title="Overextending" icon="users">
    Don't create too many Agents too quickly. Focus on quality and learning before expanding.
  </Accordion>
  <Accordion title="Ignoring Weak Agents" icon="user-slash">
    Don't abandon underperforming Agents. They may excel in different conditions or tournaments.
  </Accordion>
  <Accordion title="Poor Resource Allocation" icon="coins">
    Don't spend all resources on one Agent. Balance investments across your team.
  </Accordion>
</AccordionGroup>
